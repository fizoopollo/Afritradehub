## Customization

1. **Colors**: Edit CSS variables in `styles.css` (`:root` section)
2. **Content**: Update text in `index.html`
3. **Contact Info**: Change email, phone, address in contact section
4. **Pricing**: Modify pricing tiers in pricing section
